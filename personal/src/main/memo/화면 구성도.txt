===   OOO 학원   ===

<메뉴 - 로그아웃 상태>

로그인 - 학생 로그인, 관리자 로그인 선택 후 이메일, 비밀번호 입력

회원가입(학생만) - 아이디(이메일 형식), 비밀번호, 이름, 생년월일, 연락처를 입력받음

이메일/PW 찾기 -
	이메일 찾기 - 이름, 생년월일, 전화번호를 입력받아서 이메일를 반환받음
	비밀번호 찾기 - 이메일, 이름을 입력받아서 비밀번호를 받음(앞 5자리를 제외하고 나머진 *표시)

종료 - 프로그램을 종료함



<학생 메뉴>

강의 조회 -
	단건 조회 - 이름으로 검색 
			 이름, 기간, 최대인원, 시작여부 출력
	전체 조회 - '이름 기간(시작일~종료일) 시작여부' 형태로 출력

교직원 조회 - 
	단건 조회 - 이름으로 검색
			 이름, 정보, 담당강의 출력
	전체 조회 - '이름 - 담당강의' 형태로 출력

수강 신청 - 수강강의 이름을 입력하면 등록여부(is_regist_lecture)가 true 로 바뀜

개인정보 수정 - 비밀번호를 입력하고 진행함
			이메일, 비밀번호, 연락처를 변경할 수 있음

회원 탈퇴 - 비밀번호를 입력해서 진행하며 탈퇴 시 학생정보가 삭제됨

로그 아웃 - 로그아웃 후 메인 메뉴로 나감

종료 - 프로그램을 종료함



<관리자 메뉴>

학생 관리 -
	학생 조회 -
		단건 상세 조회 - 이름으로 검색
					 id, 이름, 이메일, 생년월일, 전화번호, 
					 가입일, 수강중인 강의, 상태 출력
		전체 조회 - 이름, 전화번호, 수강중인 강의, 상태를 출력
	탈락 처리 - 이름으 검색해서 status 값을 'FAIL'로 바꿈
	수강신청 취소 - 이름으로 검색해서 lecture_id 값을 null 로 바꾸고
				status 값을 'NONE'으로 바꿈
	상태 업데이트 - 모든 학생들의 status 값을 현재 날짜 기준으로 바꿔줌
	학생정보 강제 삭제 - 이름으로 검색해서 삭제함

수강신청 관리 - 
	수강신청 조회 -
		단건 조회 -
			학생 이름으로 조회 - id, 학생이름, 강의이름, 신청날짜, 접수여부
			강의 이름으로 조회 - id, 학생이름, 강의이름, 신청날짜, 접수여부
		여러건 조회 -
			전체 조회 - 학생이름, 강의이름, 신청날짜, 접수여부
			미접수 강의 조회 - 학생이름, 강의이름, 신청날짜
			접수된 강의 조회 - 학생이름, 강의이름, 신청날짜
	수강신청 접수 - 이름으로 검색
				is_enrolled 값이 false 일 경우 true 로 바꾼 후
				학생의 lecture_id 에 값이 들어가고
				status 값이 'SCHEDULED'로 바뀜
	수강정보 수정 - 이름으로 검색해서 강의를 바꿀 수 있음
	
강의 관리 - 
	강의 조회 -
		단건 상세 조회 - 이름으로 검색
					 id, 이름, 설명, 기간, 학생수/최대인원수, 시작여부
		전체 조회 - 이름, 기간, 학생수/최대인원수, 시작여부 출력
	강의 등록 - 이름, 설명, 시작일, 종료일, 최대인원수를 입력받음
	강의정보 수정 - 이름으로 검색
		전체 수정 - 이름, 설명, 기간, 최대인원수 수정
		이름 수정 -
		설명 수정 -
		기간 수정 -
		최대인원수 수정 -
	강의 삭제 - 이름으로 검색해서 삭제

교직원 관리 - 
	교직원 조회 -
		단건 상세 조회 - 이름으로 검색
			 		 id, 이름, 이메일, 담당강의, 정보, 고용날짜 출력
		전체 조회 - '이름 - 담당강의' 형태로 출력
	교직원 등록 - 이름, 이메일, 담당강의, 정보를 입력받음
	교직원정보 수정 - 이름으로 검색
		전체 수정 - 이름, 정보, 이메일, 담당강의 수정
		이름 수정 -
		정보 수정 -
		이메일 수정 -
		담당강의 수정 -
	교직원 삭제 - 이름으로 검색해서 삭제
	
관리자 관리(master 관리자만) - 
	관리자 조회 -
		단건 상세 조회 -
		전체 조회 -
	관리자 등록 -
	관리자정보 수정 -
	관리자 삭제 -
	
개인정보 수정 - 비밀번호를 입력하고 진행함
			이메일, 비밀번호, 연락처를 변경할 수 있음

로그 아웃 - 로그아웃 후 메인 메뉴로 돌아감

종료 - 프로그램을 종료함
